About this data

We've been using Chrome's anonymous opt-in usage statistics to count the occurrences each CSS property is animated in transitions/keyframes. There is no limit on the number of times each property is counted per page view -- a property that is animated multiple times during a single page view is counted each time it animates. The numbers on this page indicate a percentage of occurrences (across all channels and platforms) where the property is animated. Data is ~24 hrs old.
