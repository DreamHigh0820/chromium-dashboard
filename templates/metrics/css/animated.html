About this data

We've been using Chrome's anonymous opt-in usage statistics to count the number of times each CSS property is animated in a Transition or Animation. A property that is animated multiple times during a single page view is counted each time it animates. Percentages on this page indicate the frequency each property is animated relative to the other animated properties. Data is ~24 hrs old and reflects usage across all channels and platforms.
